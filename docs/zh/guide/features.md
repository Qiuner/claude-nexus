# 功能介绍

## 文件夹管理

通过拖拽、本地持久化和轻量的文件夹操作，让你的对话历史更有条理。

![新建文件夹](/assets/v2/NewFolder.gif)

## 时间线导航

长对话不再难找。固定时间线可以帮助你快速浏览对话结构、预览关键内容，并一键跳转到需要的位置。

![时间线导航](/assets/TimeLine.gif)

## 悬浮球与对话宽度调整

悬浮球提供常用工具的统一入口，其中一个面板可以把对话宽度调整到 **38rem** 到 **90rem**，默认值为 **48rem**。

![对话宽度](/assets/DialogueWidth.gif)

## 提示词库

直接在输入框工具栏中保存、搜索、编辑和插入提示词。

- 支持导入 `gemini-voyager` 导出的 `.json` 文件
- 支持导出为兼容的 `gemini-voyager.prompts.v1` 格式
- 导入时会自动跳过重复项

![提示词库](/assets/PromptLibrary.gif)

## 对话导出

在对话工具栏中将聊天记录导出为 Markdown 或 JSON。

![对话导出](/assets/DownloadChatHistory.gif)

## 语言切换

可在扩展弹窗中切换中文与 English。

![语言切换](/assets/v2/LanguageSwitch.gif)
