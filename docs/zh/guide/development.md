# 本地开发

## 启动本地开发

```bash
yarn dev:chrome
```

每次重新构建后：

1. 打开 `chrome://extensions`
2. 刷新 `claude-nexus` 扩展
3. 刷新 `claude.ai` 页面

## 技术栈

- React 19
- TypeScript 5
- Tailwind CSS 4
- Vite
- Chrome Extension Manifest V3

## 仓库常用命令

```bash
yarn dev:chrome
yarn build:chrome
yarn build:firefox
yarn docs:dev
yarn docs:build
yarn docs:preview
```
