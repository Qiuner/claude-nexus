# 快速开始

`claude-nexus` 是一个为 [claude.ai](https://claude.ai/) 补足高频使用体验的浏览器扩展。

它适合重度 Claude 用户，帮助你更好地管理导航、组织对话和复用内容。

## 它解决了什么问题

Claude 默认界面足够简洁，但高频使用后会逐渐出现一些痛点：

- 侧边栏很快就会堆满对话
- 长对话不容易快速定位重点
- 有价值的提示词很容易散落丢失
- 导出聊天记录不够方便

`claude-nexus` 用一组聚焦的增强能力解决这些问题，同时尽量不打断你原本的 Claude 使用习惯。

## 核心能力

- **文件夹管理**：整理和归类对话
- **时间线导航**：快速跳转长对话中的关键位置
- **悬浮球面板**：集中放置常用工具入口
- **对话宽度调整**：获得更舒适的阅读宽度
- **提示词库**：沉淀并复用常用提示词
- **对话导出**：导出为 Markdown 或 JSON
- **语言切换**：支持 中文 与 English
