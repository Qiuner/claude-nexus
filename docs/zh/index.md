## 为什么选择 claude-nexus？

我每天都在使用 Claude。随着对话越来越多，侧边栏会变得越来越混乱，长对话也缺少方便的跳转和整理方式。

`claude-nexus` 的目标，就是把这些日常使用中的摩擦点变成更顺手的工作流。

## 你可以做什么

- 用文件夹组织对话历史
- 用时间线快速浏览长对话
- 通过悬浮球打开常用工具
- 复用提示词库中的常用提示词
- 将对话导出为 Markdown 或 JSON
