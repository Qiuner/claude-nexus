# Getting Started

`claude-nexus` is a browser extension that adds missing quality-of-life features to [claude.ai](https://claude.ai/).

It is built for people who use Claude heavily and want better control over navigation, organization, and reuse.

## What problem it solves

The default Claude interface is clean, but frequent use creates a few pain points:

- the sidebar gets crowded quickly
- long conversations are hard to scan
- useful prompts are easy to lose
- exporting chat history is inconvenient

`claude-nexus` adds focused tools to solve those problems without changing your normal Claude workflow.

## Main capabilities

- **Folder organization** for sorting conversations
- **Timeline navigation** for jumping through long chats
- **Floating ball panels** for quick-access tools
- **Chat width control** for a more comfortable reading layout
- **Prompt library** for reusable prompts
- **Chat export** in Markdown and JSON
- **Language switch** for 中文 and English
