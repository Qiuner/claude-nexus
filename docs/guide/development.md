# Development

## Start local development

```bash
yarn dev:chrome
```

After each rebuild:

1. Open `chrome://extensions`
2. Refresh the `claude-nexus` extension
3. Refresh the `claude.ai` page

## Tech stack

- React 19
- TypeScript 5
- Tailwind CSS 4
- Vite
- Chrome Extension Manifest V3

## Repository commands

```bash
yarn dev:chrome
yarn build:chrome
yarn build:firefox
yarn docs:dev
yarn docs:build
yarn docs:preview
```
